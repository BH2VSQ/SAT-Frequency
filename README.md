# **SatFreq Doppler 卫星频率多普勒计算工具**

欢迎使用 SatFreq Doppler 工具。这是一个基于 Web 的应用，用于显示业余无线电卫星转发器的频率信息和多普勒（Doppler）修正表格，帮助操作员快速设置无线电设备。

数据源文件为 **sat.json**，您可以编辑此文件来添加、修改或删除卫星信息。

## **🌐 应用程序使用指南**

### **1\. 访问主界面**

* **浏览器访问：** 请通过您的 Web 浏览器访问主页面：  
  * https://freq.bh2vsq.cn
* **功能：**  
  * 选择左侧列表中的卫星来查看其详细信息。  
  * 主界面会根据卫星的转发器类型（FM 或 Linear）自动生成一个五段式多普勒修正表格。  
  * 对于 **Linear** 模式的卫星，您可以拖动 **频段偏移微调** 滑块来模拟在通带内调整中心频率，表格会实时更新。

### **2\. 编辑数据源 (sat.json)**

为了让所有人都能贡献和更新数据，我们提供了一个在线编辑器。

* **编辑器访问：** 请通过以下地址访问 JSON 编辑器页面：  
  * https://freq.bh2vsq.cn/editor.html  
* **编辑流程：**  
  1. 进入 editor.html 页面。  
  2. 您将看到一个文本区域，其中加载了当前 sat.json 的内容（或一个空模板）。  
  3. 请根据 [**JSON 格式说明**](https://www.google.com/search?q=%23json-%E6%A0%BC%E5%BC%8F%E8%AF%B4%E6%98%8E) 部分的规范，对数据进行编辑。  
  4. 编辑完成后，点击 **“导出 JSON 文件”** 按钮。浏览器会将编辑后的内容保存为一个名为 sat.json 的文件到您的本地电脑。

## **💡 贡献数据到 GitHub**

## **📊 JSON 格式说明**

sat.json 文件的结构是一个包含多个卫星对象的 **JSON 数组**。

### **根结构**

\[  
  { /\* 卫星对象 1 \*/ },  
  { /\* 卫星对象 2 \*/ },  
  // ... 更多卫星  
\]

### **卫星对象 (Satellite Object) 字段**

| 字段名 | 类型 | 是否必填 | 描述 | 示例 | 可接受值 |
| :---- | :---- | :---- | :---- | :---- | :---- |
| id | 字符串 | 是 | 卫星的唯一标识符（建议使用缩写，如呼号）。 | "AO-92" | 任意非空字符串 |
| name | 字符串 | 是 | 卫星的全名或常用名。 | "AO-92 (FO-29)" | 任意非空字符串 |
| noradId | 数字 | 否 | 卫星的 NORAD ID，用于 TLE 跟踪。 | 27607 | 整数 |
| notes | 字符串 | 否 | 任何操作提示或重要信息。 | "需要亚音74.4激活开机" | 任意字符串 |
| orbitType | 字符串 | 否 | 卫星的轨道类型，用于判断多普勒效应程度。 | "LEO" | **LEO** (低地轨道), **GEO** (地球同步轨道) |
| transponders | 数组 | 是 | 卫星包含的转发器列表（至少包含一个）。 | \[...\] | 转发器对象数组 |

### **转发器对象 (Transponder Object) 字段**

转发器根据 **type** 字段分为两种主要模式，必填字段略有不同。

#### **1\. FM (固定频率模式 \- Fixed Frequency Mode)**

适用于 FM 语音、APRS、SSTV 等使用固定上下行频率的模式。

| 字段名 | 类型 | 是否必填 | 描述 | 示例 | 可接受值 |
| :---- | :---- | :---- | :---- | :---- | :---- |
| id | 字符串 | 是 | 转发器的唯一标识符。 | "fm-v-u" | 任意非空字符串 |
| name | 字符串 | 是 | 转发器的友好名称。 | "V/U FM 语音" | 任意非空字符串 |
| type | 字符串 | 是 | 模式类型标识。 | "FM" | **FM**, **APRS**, **SSTV** 或其他非Linear模式 |
| uplink | 数字 | 否 | 上行频率 (MHz)。**如果只有下行 (如信标/SSTV)，可省略。** | 145.990 | 浮点数 (MHz) |
| downlink | 数字 | 否 | 下行频率 (MHz)。**如果只有上行，可省略。** | 437.800 | 浮点数 (MHz) |
| tone | 数字 | 否 | 上行亚音（CTCSS）频率 (Hz)。 | 67.0 | 浮点数 (Hz) |

#### **2\. Linear (线性模式 \- SSB/CW Mode)**

适用于 SSB/CW 等宽带转发器，频率为中心频率。

| 字段名 | 类型 | 是否必填 | 描述 | 示例 | 可接受值 |
| :---- | :---- | :---- | :---- | :---- | :---- |
| id | 字符串 | 是 | 转发器的唯一标识符。 | "lin-u-v" | 任意非空字符串 |
| name | 字符串 | 是 | 转发器的友好名称。 | "U/V 线性模式" | 任意非空字符串 |
| type | 字符串 | 是 | 模式类型标识。 | "Linear" | **Linear** |
| uplinkBase | 数字 | 是 | 上行中心频率 (MHz)。 | 435.640 | 浮点数 (MHz) |
| downlinkBase | 数字 | 是 | 下行中心频率 (MHz)。 | 145.965 | 浮点数 (MHz) |
| uplinkMode | 字符串 | 是 | 上行调制模式。 | "USB" | **USB** (Upper Sideband), **LSB** (Lower Sideband) |
| downlinkMode | 字符串 | 是 | 下行调制模式。 | "LSB" | **USB**, **LSB** |
| isInverting | 布尔值 | 是 | **重要：** 转发器是否反向（下行频率变化方向与上行相反）。 | true | **true** 或 **false** |
| bandwidth | 数字 | 是 | 转发器的总通带带宽 (kHz)。 | 30.0 | 浮点数 (kHz) |
